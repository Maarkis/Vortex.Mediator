# Support

## Getting Help

If you need help using this project, start with the existing documentation:

- [README.md](./README.md)
- [CONTRIBUTING.md](./CONTRIBUTING.md)

If the documentation does not answer your question, use the most appropriate channel:

- Open an issue for reproducible bugs, broken behavior, or documentation problems
- Open a discussion for usage questions, design ideas, or general guidance, if Discussions are enabled

## Before Opening an Issue

Please try to include:

- The .NET SDK version you are using
- The target framework involved
- A minimal reproduction
- The expected behavior
- The actual behavior
- Any relevant logs, stack traces, or generated source output

## Security Issues

Do not report security vulnerabilities through public issues.

If you identify a security issue, contact the maintainers through an appropriate private channel.

## Support Scope

Support is provided on a best-effort basis.

The maintainers may prioritize:

- Confirmed bugs
- Regressions
- Documentation gaps
- Issues that affect published packages and supported scenarios

## Commercial Support

This project does not currently provide commercial support or service-level agreements.
